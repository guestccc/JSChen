# 面向对象（oop），面向过程（POP），函数式编程（fp）

## 面向对象（oop）object oriented programming

冰箱 打开

冰箱 塞大象

冰箱 关闭

```js
冰箱.open()
冰箱.push()
冰箱.close()
```

## 面向过程（POP）procedure oriented Programming

打开冰箱

装进大象


```js
打开冰箱()
把大象装进冰箱()
关闭冰箱()
```

## 函数式编程（fp）functional programming

大象 -> 装进冰箱的大象

```js
装冰箱(大象) => 装进冰箱的大象
```

函数式编程特点

允许把函数本身作为参数传入另一个函数，还允许返回一个函数！
